# 🛍️ E-Commerce Product Catalog

## 📌 Project Overview

The **E-Commerce Product Catalog** is a full-stack web application developed using HTML, CSS, JavaScript, Node.js, and Express.js. It allows users to browse products, search items, add products to a shopping cart, remove items from the cart, calculate the total price, switch between dark and light themes, and navigate through the application without reloading the page.

## 🚀 Features

* Display 15 products with images and prices
* Product search functionality
* Add products to the shopping cart
* Remove products from the shopping cart
* Automatic total price calculation
* Client-side routing (Home, Products, Cart)
* Dark Mode / Light Mode
* Responsive design
* Backend API using Express.js

## 🛠️ Technologies Used

## ⚙️ How to Run

1. Start the backend server:

```bash
node server.js
```

2. Open `index.html` using Live Server in Visual Studio Code.
